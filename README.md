### Run Locally

To run **Swiggy** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/bhaskarnakka/Swiggy.git
```

Windows:

```bash
git clone https://github.com/bhaskarnakka/Swiggy.git
```
